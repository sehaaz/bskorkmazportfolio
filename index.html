Temperature of Premier League

This project evaluates the matches in the English Premier League. It tries to find the effect of temperature on the match result.

Github

I used Python language in this project.


I created this project with the help of object-oriented programming using libraries such as Pandas, Matplotlib, Requests (API), and Tkinter (GUI) available in Python.


This project takes the matches from the 2000-2001 season to the 2018-2019 season as a parameter. (7000+ matches)


If you want, you can take a desired interval between these seasons as a parameter yourself. You can also view the results of any team or referee you want. As an extra, you can do a simulation match according to the results you get.
